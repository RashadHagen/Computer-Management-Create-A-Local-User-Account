<h1>Computer Management – Create A Local User Account</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
This project goes over how to create a local user account in Computer Management. Creating a local user in Computer Management allows you to manage user accounts on a single computer, independent of a network or domain. This means the account will only be valid for that particular device.
</b>
